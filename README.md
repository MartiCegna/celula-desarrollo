Escenario B – Análisis de Ventas de una Pequeña Empresa

Descripción del Proyecto

El objetivo de este proyecto es analizar un conjunto de datos de ventas comerciales utilizando Python y herramientas de control de versiones.

Se implementó un flujo de trabajo basado en Git, GitHub y Jira para gestionar tareas, registrar cambios y mantener la trazabilidad del desarrollo.

Dataset Utilizado

Se utilizó un archivo CSV con información de ventas que contiene:

Fecha de venta
Producto
Cantidad vendida
Precio unitario
Funcionalidades

El script desarrollado permite:

Calcular las ventas totales.
Identificar el producto más vendido.
Analizar la información de ventas almacenada en el dataset.
